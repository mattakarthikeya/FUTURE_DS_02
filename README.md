# Customer Retention & Churn Analysis (Task 2)

## 📌 Objective
Analyze customer data to identify churn patterns, retention trends, and key factors affecting customer retention.

## 🛠 Tools Used
- Power BI
- Excel / CSV Dataset

## 📊 Dashboard Overview
This dashboard includes:
- Total Customers
- Churn Rate
- Churn by Contract Type
- Churn by Internet Service
- Customer Tenure Analysis
- Payment Method Analysis

## 🔍 Key Insights
- Customers with month-to-month contracts have the highest churn.
- Customers with low tenure are more likely to churn.
- Fiber optic users show higher churn compared to DSL users.

## 💡 Recommendations
- Provide discounts for long-term contracts.
- Improve service quality for fiber users.
- Focus on retaining new customers through engagement strategies.
